NIRCam primary / NIRISS parallel mosaic release

December 7, 2022

The reduced images from the UNCOVER pre-imaging observations (Nov 2,4,7, & 15, 2022) are now available!

The short- (F115W, F150W, F200W) and long-wavelength bands (F277W, F356W, F410M, F444W) use pixel scales of 0.02" and 0.04", respecitvely. All filters have the same WCS, which is defined so that the short-wavelength pixels can be binned 2x2 to directly match the long-wavelength images.

Additionally, mosaics of all existing HST data are provided, produced with the same WCS and a 0.04" pixel scale.

These mosaics include data from the following programs:
  • JWST/NIRCAM: GO-2561 (UNCOVER), ERS-1324 (GLASS), DD-2767
  • JWST/NIRISS: GO-2561 (UNCOVER), ERS-1324 (GLASS)
  • HST/ACS: #11689 (PI: Dupke), #13386 (PI: Rodney), #13495 (PI: Lotz / HFF), #13389 (PI: Siana), #15117 (PI: Steinhardt / BUFFALO), #17231 (PI: Treu)
  • HST/WFC3: #13495 (PI: Lotz / HFF), #15117 (PI: Steinhardt / BUFFALO)

Please cite Bezanson et al. 2022 if using these mosaics in any publication.

UNCOVER Abell 2744 lens model release

December 14, 2022

Authors: Lukas J. Furtak and Adi Zitrin

The UNCOVER strong lensing (SL) model is a parametric SL model constructed with the UNCOVER JWST data using a new (grid-free) analytic method by A. Zitrin.

In this model releases, we supply WCS-registered FITS maps of the deflection field (in pixels), convergence, shear, and the lensing potential normalized to Dds/Ds=1. We also supply magnification maps for various source redshifts: 1, 2, 4, 6, 8, 10, 15 and 20. The lensing maps have a FoV of 7.6' x 7.6', which covers the entire central UNCOVER field. We provide maps for the best-fit model in two resolutions, a high-resolution version in 0.04"/pix (JWST NIRCam resolution) and a low-resolution version with 0.1"/pix.

Please cite Furtak et al. 2022 when using any of the lensing maps provided here.

For any questions, please contact us at: furtak [at] post.bgu.ac.il

v1.1 UPDATE: July 25, 2023

New v1.1 lensing models are available! Compared to the v1.0 models, we added an additional spectroscopic redshift in the north-western sub-structure as well as six new multiple images. The additional constraints improve the model RMS down to 0.5.

Initial photometric catalog release: NIRCam

January 9, 2023

Authors: John R. Weaver, Sam Cutler, Richard Pan, Kate Whitaker, and Lukas Furtak

The initial DR1 photometric catalogs from the UNCOVER Abell 2744 NIRCam primary observations are now available!

Two catalogs are available, with different color apertures:

  • 0.32" diameter (D032), intended for compact, faint, high-z, or crowded sources; and
  • 0.7"   diameter (D070), intended for large, bright, low-z, or isolated sources

Both catalogs are based on F444W PSF-matched imaging, where the bright cluster galaxies and ICL have been modeled and subtracted. Detection was performed on a noise-equalized long-wavelength image (F277W+F356W+F444W). Initial photometric redshifts derived using EAZY, and lensing magnifications for each object based on the lensing maps of Furtak et al. 2022 are also included in the catalogs. For full details, please see the README and the catalog paper.

Please cite Weaver et al. 2023 if using these photometric catalogs in any publication.
